# Marketing & Product Performance - Machine Learning Analysis

## 📌 Problem Statement
This project aims to predict product sales performance using marketing and product-related features. The task is formulated as a supervised classification problem.

## 📊 Dataset
Source: Kaggle - Marketing and Product Performance Dataset

## 🎯 Objectives
- Evaluate baseline models
- Compare advanced ML models
- Analyze preprocessing impact
- Identify key influencing features
- Evaluate model robustness
- Recommend best model

## 📈 Research Questions
RQ1: Baseline performance  
RQ2: Model comparison  
RQ3: Effect of preprocessing  
RQ4: Feature importance  
RQ5: Metric sensitivity  
RQ6: Robustness  
RQ7: Final recommendation  

## 🛠️ Technologies Used
- Python
- Scikit-learn
- XGBoost
- Pandas
- Matplotlib

## 📁 Outputs
- Figures (PDF)
- Tables (CSV)
- Final Report

## 🚀 How to Run
1. Upload dataset to Kaggle
2. Run notebooks in order (RQ1 → RQ7)
3. Outputs saved in `/kaggle/working/`
