# Marketing ML Project

## Problem
Predict high-performing products using marketing data.

## Dataset
Kaggle dataset:
https://www.kaggle.com/datasets/imranalishahh/marketing-and-product-performance-dataset

## Task
Classification

## Models
Logistic Regression, Decision Tree, Random Forest, XGBoost, SVM

## Outputs
- CSV tables
- PDF figures

## How to Run
1. Upload dataset to Kaggle
2. Run notebook
2. Run notebooks in order (RQ1 → RQ7)
3. Outputs saved in `/kaggle/working/`
